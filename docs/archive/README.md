# archive/

Resting place per documenti congelati: snapshot point-in-time, ambienti ritirati,
piani one-off conclusi. **Nessun contenuto cancellato** -- solo raggruppato e
preservato per riferimento storico (no-loss).

## Contenuto

- `ryzen-memory-archive/` -- dump memory ambiente Ryzen precedente (~141 file). Contiene
  memory sovereign (vault + personali): escluso da `.aiderignore` e dall'ASCII-guard CI.
- `aa01-handoff/` -- handoff AA01 staging (ex `aa01-handoff/`); canonico AA01 vive in `~/aa01/`
- `plans/` -- piani di integrazione one-off conclusi (ex `plans/`)
- `BACKLOG-archive.md`, `OPEN_DECISIONS-archive.md`, `status-multi-repo-history-2026-05.md`,
  `compact-context-history-2026-05.md` -- snapshot storici dei doc-radice
- `ROADMAP.md`, `SPRINT_01.md`, `SPRINT_02.md` -- artefatti di pianificazione storici
