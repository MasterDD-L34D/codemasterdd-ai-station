## 2. Problems Found

- **Overcrowded Root:** `docs/godot-v2/` has ~80+ files mixed together.
- **Orphaned/Stale Files:** Old sprint plans and handoffs clutter the root.
- **Missing Indices:** Categories like QA, Sprints lack a `README.md`.
- **Oversized Files:** E.g. `sprint-context-archive.md` should be split.

## 3. Proposed Structure

- `docs/godot-v2/`: Root for high-level docs only (PRD, SOT).
- `docs/godot-v2/architecture/`: Engine divergency, net stack.
- `docs/godot-v2/art/`: Art drops, ferrospora, UI refs.
- `docs/godot-v2/design/`: Feature specs, design gaps.
- `docs/godot-v2/qa/`: Smoke tests, eval closures.
- `docs/godot-v2/sprints/`: Historical sprint docs.
- `docs/godot-v2/misc/`: Uncategorized / legacy files for later review.
- `docs/museum/`: Graveyard for superseded specs.
- `docs/playtest/` & `docs/research/`: Maintained for isolated work.
