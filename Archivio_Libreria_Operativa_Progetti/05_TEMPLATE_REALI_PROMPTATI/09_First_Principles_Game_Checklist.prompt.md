# Prompt Template — Compilazione FIRST_PRINCIPLES_GAME_CHECKLIST

```text
Agisci come Game Systems Designer + Principal Engineer.

Ti fornirò contesto di gioco, stato del repo, sistemi attuali e problemi percepiti.

Voglio che tu compili o mi aiuti a compilare una checklist first-principles per decidere se il progetto è pronto a essere rifattorizzato correttamente.

Compila queste sezioni:
1. verità fondamentali del gioco
2. core mechanic first
3. test di cancellazione
4. triade fondamentale del progetto
5. rule of threes / progressione di apprendimento
6. player dynamics first
7. rational design
8. implicazioni per il repo
9. decisione finale sul tipo di refactor
10. gate finale

Per ogni sezione:
- marca ciò che è chiaro
- marca ciò che è incerto
- evidenzia le ipotesi ancora non validate
- proponi cosa devo verificare prima di codificare

Se i dati sono insufficienti, non inventare. Scrivi cosa manca.

Contesto:
[incolla brief, design notes, problemi del repo, sistemi attuali]
```
