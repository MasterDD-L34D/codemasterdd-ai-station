# handoffs/

Session handoff + continuity logs. Narrativa per-sessione operativa (journal esteso),
usata per riprendere il lavoro cross-session. Ex `sessions/` (rename 2026-06-04).

Log storici, append-only. Tipicamente congelati una volta chiusa la sessione.

## Convenzione

Nome file: `YYYY-MM-DD-<slug>.md` (es. `2026-05-28-handoff-sot-drift-A.md`,
`2026-06-02-screen-scaling-godot-handoff.md`).

Per la continuita' di sessione corrente vedi anche `COMPACT_CONTEXT.md` (root)
e le memory per-progetto.
