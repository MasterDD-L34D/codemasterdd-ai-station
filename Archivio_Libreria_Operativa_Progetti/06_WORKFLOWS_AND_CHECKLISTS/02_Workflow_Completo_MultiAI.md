# Workflow Completo — NotebookLM -> ChatGPT -> Claude Code

## Step 1 — Raccolta
Metti i materiali in `00_Inbox`.

## Step 2 — Staging
Ordina il corpus almeno per:
- brief
- design core
- round/combat
- systems/economy
- UX/player experience
- backlog/decisioni
- screenshot/reference
- materiale vecchio o conflittuale

## Step 3 — NotebookLM
Ottieni:
- overview progetto
- unità minima di partita
- contraddizioni
- brief engineer
- sintesi densa

## Step 4 — ChatGPT
Trasforma tutto in:
- brief
- compact
- decision log
- backlog
- vincoli hard
- prompt successivi

## Step 5 — Claude Code
Usa il brief per:
- repo map
- audit tecnico
- first principles reconstruction
- strategia di migrazione
- primo modulo atomico

## Step 6 — Ritorno alla libreria
Riporta in archivio:
- risultati di Claude Code
- design ricostruito
- migration plan
- domande bloccanti
- prossima azione singola più utile

## Step 7 — Compact finale
Aggiorna:
- `COMPACT_CONTEXT.md`
- `DECISIONS_LOG.md`
- `BACKLOG.md`
- `REFERENCE_INDEX.md`
