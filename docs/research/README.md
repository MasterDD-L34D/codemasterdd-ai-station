# research/

Ricerche esplorative e prototipazione tecnica, datate. Documenti point-in-time
prodotti durante una decisione/indagine (bench modelli, audit, methodology).

## Convenzione

Nome file: `<topic>-YYYY-MM-DD.md` oppure `YYYY-MM-DD-<topic>.md`
(es. `bench-mixed-workload-2026-05-09.md`,
`2026-05-28-codemasterdd-first-principles-audit.md`).

Report quality-gate ricerca per componente: vedi Quality Gate globale
(`docs/research/<componente>-<date>.md`).
