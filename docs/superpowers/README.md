# superpowers/

Capabilities degli agenti AI: design specs, implementation plans, smoke test
per-agent, e documentazione Jules.

## Sottocartelle

- [`specs/`](specs/) -- design specs (cross-repo orchestrator, governor, ecc.)
- [`plans/`](plans/) -- implementation plans datati
- [`tests/`](tests/) -- smoke test log per-agent (ex `agent-smoke-tests/`)
- [`jules/`](jules/) -- Jules capabilities master + autonomy gap / dispatch (ex `jules/` + `goals/`)

Definizione subagent attivi: `.claude/agents/README.md` (root repo).
