# adr/

Architecture Decision Records -- single source of truth delle decisioni architetturali.
Formato MADR + policy ADR-0010. Authoritative: in caso di conflitto, l'ADR vince
sulle note sparse.

## Convenzione

- Nome file: `NNNN-<slug>.md` (es. `0011-cross-agent-commit-governance.md`)
- Titolo ADR: `# ADR-NNNN -- Title` (em-dash ammesso solo nei titoli ADR, eccezione ASCII-guard)
- Stato: Proposed / Accepted / Superseded

## Indice

Tabella sintetica stato/decisione: `DECISIONS_LOG.md` (root).
Ultimo ADR alla data del riassetto (2026-06-04): ADR-0039.
